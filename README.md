# ✈️ SkyLink Airways — Airline Reservation & Flight Management System

SkyLink Airways is a console-based C++ application that simulates the core functionality of a real-world airline reservation system. Developed as an Object-Oriented Programming (OOP) project, it demonstrates the practical implementation of the four fundamental OOP principles: **Encapsulation, Inheritance, Polymorphism, and Abstraction**.

The system enables users to manage flights, make reservations, book and cancel tickets, and maintain passenger records through an interactive command-line interface. Its modular design emphasizes code reusability, maintainability, and scalability while modeling real airline operations.

## 🚀 Features
- Flight scheduling and management
- Passenger registration and record management
- Ticket booking and cancellation
- Flight search and availability checking
- Administrative controls for managing airline operations
- Object-oriented architecture with modular class design

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## 🎯 Project Objective
This project was built to apply object-oriented design principles to a real-world scenario and demonstrate how OOP can be used to develop scalable, maintainable, and efficient software systems.
